# 海洋物联网主题改版方案

## 一、项目定位
当前平台定位由“通用边缘物联网平台”进一步明确为：

**海洋物联网监控平台 / Ocean Edge IoT Command Platform**

适用场景：
- 海洋环境监测
- 海域边缘站点接入
- 海上视频监控
- 浮标 / 传感器 / 海域感知设备管理
- 海洋边缘节点与数据链路监控

---

## 二、整体业务流程

### 1. 登录入口
用户进入登录页：
- 用户登录
- 新用户注册

登录或注册成功后统一跳转：
- `指挥大屏 / bigscreen.html`

### 2. 指挥大屏
指挥大屏作为平台主界面。

普通用户：
- 只能查看大屏内容
- 不显示后台管理入口

管理员：
- 查看大屏内容
- 右上角显示“后台管理”按钮
- 可进入后台管理系统

### 3. 后台管理
后台管理仅管理员可访问。

后台包含：
- 用户管理
- 设备管理
- 网关管理
- 遥测监控
- 视频管理
- 系统状态
- 返回指挥大屏

---

## 三、界面层次结构

### 1. 登录页（简洁）
目标：
- 极简
- 海洋风格
- 只承担身份入口功能

内容：
- 平台名称
- 一句海洋物联网定位文案
- 登录 Tab
- 注册 Tab
- 登录成功直达大屏

### 2. 指挥大屏（主界面）
目标：
- 海洋监控中心
- 视频 + 遥测 + 状态汇总
- 作为平台主要使用界面

内容：
- 海域总览指标
- 视频矩阵
- 海洋遥测趋势
- 最新感知流
- 视频链路状态
- 当前登录用户信息
- 管理员后台入口

### 3. 后台管理（管理员）
目标：
- 运营 / 运维 / 配置后台
- 收纳用户、设备、网关、视频、系统等功能

内容：
- 后台首页摘要
- 用户管理
- 设备管理
- 网关管理
- 遥测监控
- 视频管理
- 系统状态
- 返回指挥大屏

## 九、开发顺序建议

### 第一阶段：统一主题
- 登录页海洋化
- 指挥大屏海洋化
- 后台页海洋化

### 第二阶段：统一文案与卡片命名
- 大屏标题与指标名称海洋化
- 后台菜单与说明海洋化

### 第三阶段：后台首页内容化
- 增加后台首页摘要卡片
- 增加快捷入口矩阵

### 第四阶段：继续功能扩展
- 告警中心
- 遥测图表增强
- 视频播放位增强
- 审计日志

---

## 十、当前结论
后续整套界面不再走通用工业风，而是统一为：

**海洋物联网监控中心风格**

并遵循：
- 登录页简洁
- 登录后进入指挥大屏
- 管理员从大屏进入后台
- 后台承载全部管理功能
- 整体视觉统一为深海监控中心主题
